# DLC - Debian Linux Clear
一款基于纯C语言编写的Linux系统深度清理工具，专为 Debian/Ubuntu/Kali/LinuxMint 等Debian系发行版打造，安全无风险，绝不清理内核文件，需管理员权限运行。

## 核心特性
- 纯C编写，无任何第三方依赖，编译速度快，运行高效轻量化
- 深度全量清理：APT无用依赖、APT缓存、系统日志、全局临时文件、所有用户缓存、回收站、缩略图、系统级冗余缓存
- 安全防护：仅清理系统冗余垃圾文件，过滤内核相关目录，无系统崩溃风险
- 代码规范无冗余，0编译警告，纯中文友好提示，操作简单无门槛
- 支持编译直接运行、一键打包deb全局安装两种使用方式

## 使用说明（完整集成，所有命令复制即用）
### 方式一：编译后直接运行（推荐，极简无依赖，无需安装）
```bash
# 进入源码目录
cd dlc
# 编译源码
gcc dlc.c -o dlc -O2 -Wall
# 添加执行权限
chmod +x dlc
# 运行深度清理（必须管理员权限）
sudo ./dlc
